#### 功能实现：

1. 提取出一张图片的R、G、B三个通道的值并且将其分别转换为十进制数据存储到R.txt、G.txt、B.txt文档中
2. 再将R.txt、G.txt、B.txt里的十进制数据恢复成图片形式，用以验证

#### 文档介绍：

- libjpeg是提取R、G、B三个通道数据的代码
  - QT中c++实现
- recover.py是恢复成图片的代码
  - python实现
